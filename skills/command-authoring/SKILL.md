# Command Authoring

Use this skill when the task centers on design concise command files with predictable structure and purpose.

Core workflow:
1. Clarify the desired behavior or repo outcome.
2. Inspect the relevant files, metadata, or prior context.
3. Make the smallest coherent changes that preserve consistency.
4. Validate the new behavior or scaffold shape.
5. Summarize what changed and any remaining assumptions.
